# @36node/sketch

[![version][0]][1] [![downloads][2]][3]

Sketch is a baseground of 36node team. We use these framework to build our service and app.

## Usage

```sh

// init project
npx @36node/sketch init some-folder-name

// start service or web app
sketch start

// test
sketch test

// build
sketch build
```
