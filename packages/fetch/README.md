# @36node/fetch

[![version][0]][1] [![downloads][2]][3]

## Usage

```bash
yarn add @36node/fetch
```

```js
import fetch from "@36node/fetch";

const res = await fetch("some url", { headers: {} });
console.log(res.headers);
console.log(res.body);
```
