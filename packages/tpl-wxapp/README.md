# @36node/tpl-wxapp

微信小程序框架

## 技术框架

- 使用[Taro](https://taro.aotu.io/)
- 使用[Taro UI](https://taro-ui.aotu.io/)组件库
- 遵循 React 语法规范
- 任何问题请查看官方文档

## Usage

1. `yarn install`
2. dev: `yarn dev:weapp`
3. build: `yarn build:weapp`
